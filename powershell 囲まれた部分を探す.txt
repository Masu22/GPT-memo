PowerShellでテキストファイルから特定の文字に囲まれた部分を抽出するには、正規表現を使用するのが一般的です。例えば、`<`と`>`に囲まれた文字列を抽出したい場合、次のようにできます。

### 例: `<` と `>` に囲まれた部分を抽出

```powershell
# ファイルを読み込む
$fileContent = Get-Content -Path "C:\path\to\your\file.txt" -Raw

# 正規表現で抽出
$matches = [regex]::Matches($fileContent, "<(.*?)>")

# 抽出された結果を表示
$matches | ForEach-Object { $_.Groups[1].Value }
```

### 説明:
1. `Get-Content -Raw` でファイルの内容を1つの文字列として読み込みます。
2. `[regex]::Matches` を使って、`<` と `>` に囲まれた部分を正規表現で検索します。`<(.*?)>` というパターンは、`<` と `>` の間の任意の文字列（最短一致）をキャプチャします。
3. 抽出されたマッチング結果は `$matches` に格納されます。
4. `ForEach-Object` で各マッチを順に取り出し、囲まれた文字列（`Groups[1].Value`）を表示します。

### 例: `[` と `]` に囲まれた部分を抽出

もし別の文字（例: `[` と `]`）で囲まれた部分を抽出したい場合、次のようにパターンを変更します。

```powershell
$matches = [regex]::Matches($fileContent, "\[(.*?)\]")
```

この方法で、PowerShellを使って任意の文字に囲まれた部分を抽出することができます。
